# Animated Minecraft logo

Download [here](https://modrinth.com/mod/animated-loading-screen)!

This Minecraft mod animates the Mojang loading logo, using the animation from Minecraft Dungeons.

Original idea is from [here](https://www.curseforge.com/minecraft/mc-mods/animated-splash-screen), rewritten from scratch to work with modern Minecraft

Starting from v2.0 this mod uses stonecutter and supports from 1.19.x to 26.1.x, and forge, fabric & neoforge!

Most of the code is in SpashOverlayMixin.java, a single-file mostrosity LOL. For 2.0 [stonecutter template](https://github.com/rotgruengelb/stonecutter-mod-template) was used, though I wrote the two .unobf Gradle scripts for 26.1.x and fixed a few bugs.

Thanks to @schemil053 for adding the sound!
